# apache-htaccess
Test htaccess rules

## Run
```
docker-compose up
```

Then go to `localhost:8000` (or `192.168.99.100:8000` for Docker Toolbox)

## Regular expression tester

Go to `localhost:8000/regexptester.php` (or `192.168.99.100:8000/regexptester.php` for Docker Toolbox)
